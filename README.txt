Mod to help simulate dual screen support with a windowed game that is stretched across two monitors

To set which screen the UI is moved to, enter /leftMonitor or /rightMonitor into the chat window
To set whether the game automatically splits the screen, enter /enableAutoSplit or /disableAutoSplit into the chat window

Install mod to %USERPROFILE%\Documents\My Games\Gas Powered Games\Supreme Commander Forged Alliance\mods\
